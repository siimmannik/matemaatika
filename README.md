# 7. klassi matemaatika sisseastumiseksami harjutused

Interaktiivne veebileht 7. klassi matemaatika sisseastumiseksamiks harjutamiseks.

## Funktsionaalsus

*   **Teooria:** Ülevaatlikud teemad ja reeglid.
*   **Testid:** Enesekontrolli testid tagasisidega.
*   **Harjutused:** Praktilised ülesanded lahenduste kontrolliga.
*   **Mänguline element:** Maskott, progressiriba ja konfetid edusammude tähistamiseks.

## Tehnoloogiad

*   HTML5
*   CSS3 (Kaasaegne disain, animatsioonid)
*   JavaScript (Interaktiivsus)
